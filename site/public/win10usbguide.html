Win 10 USB Guide

  1. Download the Windows 10 ISO from here (don't worry, it looks dodgy but it's an official Microsoft link)
    • https://software-download.microsoft.com/db/Win10_20H2_v2_English_x64.iso?t=73a306fa-9182-460a-8cf8-188599fa198b&e=1608406965&h=f128ca0cf63680218a79fd8750ac5f36
    • This should link to a ~5GB .iso file, if this link doesn't work send me a message on Whatsapp
  2. Plug the USB that you want to load Windows on to into your PC
  3. Go to https://rufus.ie/
  4. Scroll down to "Download" and select the first link
  5. Run the .exe you just downloaded (it's probably called something like "rufus-3.12.exe")
  6. Select the USB you plugged in from the drop-down
  7. To the right of the "Boot selection" drop-down, click on the select button
  8. Find and select the Windows ISO you downloaded in step 1
  9. Under "Format Options", you can choose the new name of the USB (or just leave it as is)
  10. Don't change any of the other options and then click "START" at the bottom
  11. It will come up with a warning, this is just to double check you have the right USB device selected
  12. Wait until the bar at the bottom turns green (this might take a while) and it says "READY"
  13. Press the "CLOSE" button and then unplug your USB
  14. Plug the USB into the machine you want to install Windows on and get it to boot from that USB
  15. Install Windows!
